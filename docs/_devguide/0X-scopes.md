## Scopes

Emc4j rely heavily on dependency injection. So it uses classic DI scopes like "Singleton" 
and "Prototype" but it also uses a custom made scope, the 
[DocumentScope]({{ site.baseurl }}/apidocs/emc4j.core/provide urlcom/treilhes/****.html){:target="_blank"}.
